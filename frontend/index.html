Strategy Overview

The model predicts short-term volatility direction using an LSTM classifier trained on realized volatility features. When volatility is expected to rise, portfolio exposure is reduced. When volatility is expected to fall or remain calm, exposure is increased.

This implements volatility-targeted position sizing — a standard technique in systematic risk management. The strategy is always long. No shorting. No leverage beyond 1.5×.

Signal logic: vol_prob > 0.6 → reduce to ~0.5–0.9× · vol_prob < 0.4 → increase to ~1.1–1.5× · else hold current position. Positions smoothed with 5-day EMA. Transaction cost 0.02% per unit change.
